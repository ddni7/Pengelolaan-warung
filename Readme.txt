Cara Instalasi:
1. Extraks File/namafolder.
2. Buka Xampp dan klik start Apache dan Mysql, lalu buka phpmyadmin.
3. Buat Database dengan nama 'warungaman'.
4. Lalu tekan import, choose file.
5. Pada browser ketik 'localhost/namafolder'.
6. Lalu login dan coba pilih menu yang ada di halaman home tersebut.
